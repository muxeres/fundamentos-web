Sobre a atividade

A Atividade somativa 1, tem como objetivo evidenciar nosso aprendizado na disciplina de Fundamentos de Programação Web.
Colocando em prática os conhecimentos adquiridos na disciplina, atravéz do código HTML. Na atividade, foi criado um site
onde o usuário pode fazer operações aritméticas. Utilizando o CSS, fizemos a formatação do layout, modificando as fontes,
cores e tamanhos. Atravéz do Java Script, foi criado a parte do código referente às funções e métodos para a execução dos
elementos, por exemplo o método GET, que foi utilizado para a inserção dos dados para que os cálculos sejam feitos.
